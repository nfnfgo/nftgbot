# nftgbot

# NF工具小人使用说明

简介版本*v 0.1.1*

简介：
本机器人为nf无聊时光之中制作的机器小人，有一点点基本的功能，希望能方便大家！

命令使用方法：
/acgp
发送后，机器人会随机返回一张动漫图片！大约每10秒会更新一张图片！

/sitestatus
后接网址，发送后将会返回网站的HTTP状态码以及相关说明
若网址不包含http协议标识，机器人将默认补全https://

/html
后接网址，发送后将会返回网站的html文字内容
若网址不包含http协议标识，将默认补全https://

/inithtml
后接网址，发送后将会返回网站的html内容
若网址不包含http协议标识，将默认补全https://

/timestamp
若直接发送，则返回发送时时间的时间戳
若后接时间，（格式：YYMMDDHHMMSS 例：202001011200）则返回此时间对应的时间戳
若后接时间戳，则返回该时间戳对应的时间

/md5
后接文本，返回该文本的MD5值

/myid
返回您的账户ID
