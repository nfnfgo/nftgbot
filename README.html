NF机器小人-使用文档


简介版本 Ver 0.5.0 Alpha


简介：

本机器人为nf无聊时光之中制作的机器小人，有一点点基本的功能，希望能方便大家！

nf官方频道 https://t.me/nf_share


命令使用方法：

/acgp

发送后，机器人会随机返回一张动漫图片！


/shorturl

后接网址，返回发送网址的短链接

若网址不包含http协议标识，机器人将默认补全https://


/wo2022

返回东奥奖牌榜

可用参数：-fr

含义为ForceRefresh，强制重新获取奖牌榜的信息

使用例子：

/wo2022 -fr


/qrcode

后接内容，生成包含内容的二维码。若没有添加颜色参数则默认黑色

可用参数：

color 二维码颜色，不输入该参数则默认黑色

bk 背景颜色，不输入该参数则默认白色

使用例子：

/qrcode 你好！ color-red bk-blue


/bts

后接关键词，返回关键词bt搜索结果

可用参数：

eng 切换搜索引擎，默认为bthub

p 页码，默认为1

使用例子：

/bts 开端 eng-btdigg p-2

Tips：使用/bteng 可查看可用的搜索引擎


/bili

后接网址，解析该网址的视频信息，支持bilibili，YouTube等主流平台

可用参数：-raw 返回请求的原始json信息

使用例子：

/bili https://b23.tv/b9rnM76 -raw


/tel

后接手机号码，返回手机号的相关信息


/hot

聚合热搜，返回微博，知乎，抖音的热搜


/ip

后接IPv4地址，返回IP相关信息


/save

后接网址，使用waybackmachine项目存档网页


/drifter

后接漂流瓶内容，将发送一个漂流瓶


/getdrifter

从海里捞起一个漂流瓶～


/reply_drifter

后接漂流瓶id和回复内容，回复一个漂流瓶。格式为：

/reply_drifter 漂流瓶id 回复内容


/wiki

后接关键词，返回Wikipedia中关于关键词的内容。可用参数 lang，可以指定Wikipedia的语言，示例：

/wiki telegram lang-en

上述指令指定了英文为搜索语言


/one

获取一言


/dns

后接域名（不要加入http协议头），返回域名的dns信息


/sitestatus

后接网址，返回网站的HTTP状态码以及相关说明

若网址不包含http协议标识，机器人将默认补全https://


/html

后接网址，发送后将会返回网站的html文字内容

若网址不包含http协议标识，将默认补全https://


/inithtml

后接网址，发送后将会返回网站的html内容

若网址不包含http协议标识，将默认补全https://


/timestamp

若直接发送，则返回发送时时间的时间戳

若后接时间，（格式：YYMMDDHHMMSS 例：202001011200）则返回此时间对应的时间戳

若后接时间戳，则返回该时间戳对应的时间


/md5

后接文本，返回该文本的MD5值


/myid

返回您的账户ID
